# Changelog

## 1.0.0 - 2026-07-06

Initial publication-ready release.

### Added

- Unified Streamlit page structure across the toolkit.
- Shared UI, parsing, formatting, and statistical helper modules.
- Synthetic TriNetX-like examples for Measures of Association, Kaplan-Meier, and Baseline Patient Characteristics workflows.
- Parser and statistics tests.
- Repository-level documentation, license, citation metadata, and GitHub Actions workflow.

### Changed

- Standardized terminology for input, cohort labels, analysis settings, formatting, downloads, and verification.
- Corrected effect-size plotting semantics by separating ratio-scale and standardized-effect-scale null values.
- Improved Kaplan-Meier parsing and error handling.

### Notes

This release uses synthetic examples only. Real TriNetX exports should not be committed to the repository.
